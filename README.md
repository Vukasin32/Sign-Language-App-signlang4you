# Sign-Language-App-signlang4you
Sign Language App signlang4you is web application for interactive learning of symbols for letters from American Sign Language (ASL). Beside learning sign language symbols, user can also send message in sign language to someone in a form of video. The message represents video session where user creates messages (such as "Hello", "Good day", "Love you" ...) by concatenating symbols of letters.
Technologies that were used:
- Python
- Flask
- HTML/CSS
- JavaScript
- OpenCV
- Mediapipe
- Scikit-learn
- numpy
- OS
